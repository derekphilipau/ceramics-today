The Glaze Calculator is a glaze calculation program for Windows from the UK. The site has an extensive faq sheet, news, bug reports and a BBS with questions and answers by users of the program. It is available in 16 bit and 32 bit versions for Win 3.1 and Win 95. There are also data files of British and Japanese raw materilas available, which could be useful in the conversion of glaze recipes.

Insight 5 is a classic glaze calculation program, now available for Mac, Windows 3.1 and the Win 95 platform. The web site includes downloadable demos and manuals. A HREF="http://digitalfire.com/4sight/index.html">Foresight is an "industrial strength stand-alone DOS relational database system that maintains unlimited materials, oxides, recipes, test procedures, test results, physical property, reference, and people data in the open Foxpro file and index format." It will run on DOS and Windows. These programs require only low end machines to run on - a 286 will do! There is also a Foresight manual available in Acrobat Reader format.

f you are using DOS or Windows and don't want to spend any money, check out Dr Hoggs free Glaze Software for DOS or the Zakin Glaze Software, also for DOS, by Richard Zakin.

Last but not least, check out Ceramics Today Glaze Page, a database of free glazes for you to copy and use.
